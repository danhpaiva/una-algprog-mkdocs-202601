### Passo a passo (algoritmo)

1. O fazendeiro leva **a cabra** para o outro lado do rio.
2. O fazendeiro volta **sozinho**.
3. O fazendeiro leva **o lobo** para o outro lado.
4. O fazendeiro traz **a cabra de volta**.
5. O fazendeiro leva **a couve** para o outro lado.
6. O fazendeiro volta **sozinho**.
7. O fazendeiro leva **a cabra** novamente para o outro lado.

✅ Resultado: todos atravessam em segurança, sem que o lobo coma a cabra ou a cabra coma a couve.
